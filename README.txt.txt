This tool extracts publication data for faculty members from:

1. Google Scholar
2. Scopus

No installation required.

------------------------------
HOW TO USE
------------------------------

1. Double click FacultyScraper.exe
2. Select an output folder
3. Enter faculty details:

   For Google Scholar:
   - Full name (e.g., "V Baby")

   For Scopus:
   - First Name
   - Last Name

4. Click:
   - "Scrape Scholar"
   - "Scrape Scopus"
   OR
   - "Scrape Both Sources"

5. Excel files will be generated in the selected folder.

------------------------------
REQUIREMENTS
------------------------------

- Windows 10 or above
- Google Chrome installed
- Internet connection

------------------------------
SCOPUS LOGIN
------------------------------

For Scopus scraping:
You must log in manually when the browser opens.

------------------------------
OUTPUT FILES
------------------------------

The tool generates:

- scholar_<name>.xlsx
- scopus_<name>.xlsx
